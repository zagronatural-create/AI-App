# Operations Dashboard Specification

## Panels
1. Supplier Risk Heatmap
- Rows: suppliers
- Columns: delay, quality fails, rejection, volume variance
- Color bands: low/medium/high

2. Batch Risk Matrix
- X-axis: predicted deviation probability
- Y-axis: impact score (volume x market criticality)

3. Compliance Status Panel
- Counts: PASS/WARNING/FAIL by market standard
- Critical deviations list with drilldown

4. Multi-Standard Comparison Table
- Columns: Parameter, Batch Value, FSSAI, EU, Codex, HACCP internal, Status
- Filters: batch, SKU, date, market

5. Recall Simulation Tool
- Batch search
- Impact graph + customer list + estimated spread

6. Audit Export Center
- One-click PDF/CSV generation
- report hash, generated by, timestamp, version history

## Non-Functional
- First meaningful load < 2s for latest dashboard snapshot
- All critical actions auditable
- Role-based visibility by function (QA, compliance, operations, auditor)
